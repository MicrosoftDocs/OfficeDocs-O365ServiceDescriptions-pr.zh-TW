# <a name="microsoft-365-education"></a>Microsoft 365 教育版

Microsoft 365 位在各種可滿足組織的需求的最佳方案。如果您正在尋找 Microsoft 365 與 Office 365 教育版方案之間的差異，本文將告訴您哪些功能都包含在這些項目的。 
  
Microsoft 365 提供的完整的系統，包括 Office 365、 Windows 10 和 Enterprise 行動性及安全性。下表列出 Office 365 教育版 A1、 A3 和 A5 功能以及對應 Microsoft 365 教育版 A3 及 A5 功能。若要比較 Office 365 功能跨商業及企業計劃[比較 Office 365 商務計劃](https://go.microsoft.com/fwlink/?linkid=799177)，請參閱 ＜ 或者，如需更詳細的功能清單，請參閱在[Office 365 服務說明](https://go.microsoft.com/fwlink/?linkid=839510)相關的服務說明。若要搜尋的支援文章和資訊，請參閱[Office 說明及訓練](https://support.office.com/)。

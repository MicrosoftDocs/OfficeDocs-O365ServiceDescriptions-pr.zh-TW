# <a name="microsoft-365-education"></a>Microsoft 365 教育版

您可以在各種方案中使用 Microsoft 365, 以最符合貴組織的需求。 如果您正在尋找 Microsoft 365 與 Office 365 教育版方案之間的差異, 本文將會顯示每個功能所包含的功能。 
  
Microsoft 365 提供完整的系統, 包括 Office 365、Windows 10, 以及企業行動性和安全性。 下表列出適用于教育版 A1、A3 和 A5 功能的 Office 365, 以及對應的 Microsoft 365 教育版 A3 和 A5 功能。 若要跨商務和企業方案比較 Office 365 功能, 請參閱[比較 office 365 For business 方案](https://go.microsoft.com/fwlink/?linkid=799177); 如需更詳細的功能清單, 請參閱[Office 365 服務說明](https://go.microsoft.com/fwlink/?linkid=839510)中的相關服務說明。 若要搜尋支援文章和資訊, 請參閱[Office 說明和訓練](https://support.office.com/)。

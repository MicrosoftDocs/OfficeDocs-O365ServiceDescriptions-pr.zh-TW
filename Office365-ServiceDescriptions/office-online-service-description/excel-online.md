# <a name="excel-online"></a>Excel Online

Excel Online (先前稱為 Excel Web App) Microsoft Excel 將您功能延伸至網頁瀏覽器，讓您可以使用活頁簿儲存活頁簿的網站上直接。所有的 Office 365 客戶可以檢視和限制性編輯 Office 檔案使用 Office Online。
  
> [!NOTE]
> 無法在 Excel Online 中從 Microsoft SharePoint Online 中檢視超過 30 mb 的活頁簿。若要檢視較大的檔案，您將需要在電腦上安裝 Excel 桌面應用程式。 
  
> [!NOTE]
> 為協助您比較各產品項目，本文章中所列的進階功能包括：僅適用於 Office 套件 (例如：Office Professional Plus、Office Standard 或 Office 365 ProPlus) 的 Excel 桌面應用程式功能。若要查看所有 Office Online 功能 (例如 Word Online 和 PowerPoint Online) 的清單，請參閱＜[各 Office Online 計劃中可用的功能](office-online-service-description.md#feature-availability-across-office-online-plans)＞
  
若要查看 Excel Online 所支援的檔案類型，請參閱 [Office Online 支援的檔案類型](office-online-service-description.md#supported-file-types-for-office-online)。若要深入瞭解 Excel Online 和 Excel 桌面應用程式之間的差異，請參閱[在瀏覽器和 Excel 中使用活頁簿的差異](https://go.microsoft.com/fwlink/?linkid=390875)。
  
## <a name="add-hyperlinks"></a>新增超連結
<a name="bkmk_Addhyperlinks"> </a>

## <a name="slicer-viewing"></a>交叉分析篩選器檢視

按一下 [交叉分析篩選器] 按鈕，即可檢視工作表中的交叉分析篩選器及篩選資料。您無法在 Excel Online 中建立或編輯交叉分析篩選器。
  
## <a name="sort-and-filter-data"></a>排序和篩選資料
<a name="bkmk_Sortfilterdata"> </a>

在 Excel Online 中，您可以檢視和重新排序所有工作表、排序和篩選資料，以及深入檢視 PivotTables 的詳細資料。
  
## <a name="spreadsheet-audit-and-compliance"></a>試算表審核和法規遵循
<a name="bkmk_Spreadsheetauditcompliance"> </a>

在 Microsoft Excel 桌面應用程式中，您可以使用工作表的查詢和比較功能來比較活頁簿版本、分析活頁簿的問題或不一致情況，或查看活頁簿和工作表之間的連結。Excel Online 不支援這些功能。進一步了解[如何使用試算表查詢功能](https://go.microsoft.com/fwlink/p/?LinkId=271677)。
  
## <a name="status-bar-aggregates"></a>狀態列彙總
<a name="bkmk_StatusBarAggregates"> </a>

當您在 Excel Online 中選取一組儲存格時，可以在狀態列看見 SUM、AVERAGE 及 COUNT。您也可以選擇想要查看的彙總來自訂狀態列。
  
## <a name="surveys"></a>問卷調查
<a name="bkmk_Surveys"> </a>

使用「問卷」將使用者可以填寫的 Excel Online 表單傳送給使用者。回應會自動在線上工作表進行編譯。進一步了解 [Excel 中的問卷 ](https://go.microsoft.com/fwlink/p/?LinkId=391470)。
  
## <a name="tell-me"></a>操作說明搜尋
<a name="bkmk_Tell_Me_Excel"> </a>

當您需要在 Excel Online 中完成某些作業但是不知道如何執行時，您可以使用「操作說明搜尋」搜尋功能，快速找出您想要的資訊。「操作說明搜尋」了解您正在嘗試完成的作業，並且會提供建議，協助您更快速地完成作業。
  
## <a name="undo-and-redo"></a>復原和取消復原
<a name="bkmk_UndoRedo"> </a>

Excel Online 會自動儲存您的工作。如果您編輯出錯，可使用 [復原] 或同時按下 Ctrl + Z，或者使用 [取消復原] 或同時按下 Ctrl + Y。進一步了解 [Excel Online 中的鍵盤快速鍵](https://go.microsoft.com/fwlink/p/?LinkId=272943)。
  
## <a name="vba-and-macro-scripting"></a>VBA 和巨集指令碼
<a name="bkmk_VBAMacrosscripting"> </a>

Microsoft Excel 桌面應用程式是功能相當強大的工具，可用於操控、分析和呈現資料。儘管有這麼多功能，有時候您的組織可能會發現使用 Visual Basic for Applications (VBA) 這種程式語言來執行操作更簡單，透過 VBA 建立巨集可執行一般的重複性工作，或執行使用者介面 (UI) 無法解決的部分工作。您無法在 Excel Online 中使用 VBA 建立巨集，但是可以開啟和編輯已啟用 VBA 功能的試算表，而不會移除 (或損毀) 此檔案所含的 VBA。進一步了解[開始使用巨集](https://go.microsoft.com/fwlink/p/?LinkId=271678)。
  
## <a name="view-3d-charts"></a>檢視 3D 圖表
<a name="bkmk_View3dcharts"> </a>

在您的工作表中檢視 3D 圖表。您無法在 Excel Online 中建立或編輯 3D 圖表。
  
## <a name="what-if-analysis-tools"></a>What if 分析工具
<a name="bkmk_WhatIfAnalysis"> </a>

不適用於 Excel Online。您可以如預期在 Excel Online 中檢視「如果」分析結果。如果您想使用分析工具 (例如：目標搜尋、資料表、規劃求解和數列)，則您需要 Excel 桌面應用程式。進一步了解[在瀏覽器中和 Excel 桌面應用程式中使用活頁簿的差異](https://go.microsoft.com/fwlink/p/?LinkId=273900)。

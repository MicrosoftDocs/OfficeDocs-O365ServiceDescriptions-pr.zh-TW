# <a name="powerpoint-for-the-web"></a>Web 版 PowerPoint

PowerPoint for the web (先前稱為 PowerPoint Web App) 可將您的 Microsoft PowerPoint 體驗延伸至網頁瀏覽器, 讓您在儲存簡報的網站上直接使用簡報。 Microsoft Office 365 客戶與 Office 網頁版的客戶可以在旅途中查看、建立及編輯檔案。
  
The PowerPoint Editor is a web front-end component that creates a browser-based editing surface, which enables users to work on documents without losing fidelity.
  
如果您按一下 [web 工具列 PowerPoint] 上的 [**在 powerpoint 中開啟**] 按鈕, 簡報會在 powerpoint 桌面應用程式中開啟 (如果電腦上已安裝 Microsoft PowerPoint 2010 或更新版本)。 
  
了解如何[在電腦上使用商務用 Office 365 下載及安裝 Office](https://support.office.com/en-us/article/Download-and-install-Office-using-Office-365-for-business-on-a-PC-72977511-dfd1-4d8b-856f-405cfb76839c)。
  
## <a name="alignment-bullets-numbers"></a>對齊方式、項目符號、編號
<a name="bkmk_AlignmentBulletsNumbers"> </a>

透過使用 PowerPoint 的網頁版, 您可以變更段落對齊方式、套用專案符號或編號, 以及變更專案符號或編號文字的層級。 進一步了解[在瀏覽器中和 PowerPoint 桌面應用程式中使用簡報的差異](https://go.microsoft.com/fwlink/?LinkId=272763)。
  
## <a name="apply-basic-transitions-and-animations"></a>套用基本切換和動畫效果
<a name="bkmk_Applybasictransitions"> </a>

從動畫和切換效果的圖庫中選擇。 Web 版 PowerPoint 支援八個轉換和37動畫。 PowerPoint 不支援的其他動畫和轉換會保留在簡報中, 並顯示在投影片放映中, 但無法在 PowerPoint 中針對 web 進行修改。 進一步了解[在瀏覽器中和 PowerPoint 桌面應用程式中使用簡報的差異](https://go.microsoft.com/fwlink/?LinkId=272763)。
  
## <a name="apply-rich-formatting-to-text-shapes-and-pictures"></a>將豐富格式套用至文字、形狀和圖片
<a name="bkmk_Applyrichformatting"> </a>

無法在 PowerPoint for web 中使用。 PowerPoint 桌面應用程式提供進階的格式化功能 (例如：顏色、漸層、滴管、效果和樣式)。 透過使用 PowerPoint 的網頁版, 您可以新增文字方塊或從圖案庫中選擇, 並套用定義填滿、大綱和陰影效果的樣式。 
  
## <a name="apply-themes-and-theme-variants"></a>套用佈景主題和佈景主題變化
<a name="bkmk_ApplyThemes"> </a>

新增投影片至簡報時，可從內建的佈景主題和變化圖庫中選擇想要的項目。 PowerPoint 不支援 web 的主題會保留在簡報中, 但在 PowerPoint 中無法使用修改主題的功能。 進一步了解[在瀏覽器中和 PowerPoint 桌面應用程式中使用簡報的差異](https://go.microsoft.com/fwlink/?LinkId=272763)。

<a name="bkmk_Offlineviewingandauthoring"> </a>

<a name="bkmk_Pictures"> </a>

## <a name="presenter-view"></a>簡報者檢視
<a name="bkmk_PresenterView"> </a>

無法在 PowerPoint for web 中使用。 僅 Microsoft PowerPoint 桌面應用程式可讓簡報者在幕後控制簡報流程、備註、註釋和縮放工具。 Web 版 PowerPoint 不支援這些功能。

您可以使用內建的字典搭配 web 來檢查拼寫及設定校對語言。 但 PowerPoint 網頁版不會使用自訂字典, 也不包含轉譯或同義字。
  
## <a name="real-time-co-authoring"></a>即時共同撰寫
<a name="bkmk_Coauthoring"> </a>

<a name="bkmk_AdvancedReviewerTools"> </a>

無法在 PowerPoint for web 中使用。 僅 PowerPoint 桌面應用程式提供進階審閱功能 (例如：合併衝突和比較簡報)。 透過使用 PowerPoint 的網頁版, 您可以查看、新增、編輯或刪除批註。
  
## <a name="rights-management-apply-and-consume-irm-and-password-protection"></a>版權管理：套用和使用 IRM 和密碼保護功能
<a name="bkmk_CreationPasswords"> </a>

無法在 PowerPoint for web 中使用。 [! 注意] 網頁版 PowerPoint 會顯示受資訊版權管理 (IRM) 保護的簡報。 不過, 這些簡報無法在瀏覽器中編輯, 而且您無法在 PowerPoint for web 中建立受 IRM 保護的簡報。 PowerPoint 網頁版無法開啟以密碼數位簽署或加密的簡報。

## <a name="slide-show"></a>投影片放映
<a name="bkmk_Slideshow"> </a>

網頁版 PowerPoint 會在全螢幕視窗中播放投影片放映。 按下空格鍵可前往下一張投影片。 投影片可播放動畫，但僅支援投影片之間的 [淡化] 和 [擦去] 切換效果。 進一步了解[在瀏覽器中和 PowerPoint 桌面應用程式中使用簡報的差異](https://go.microsoft.com/fwlink/?LinkId=272763)。

<a name="Bkmk_TellMe_PPT"> </a>

當您需要在 PowerPoint 中為網頁完成某些作業但不知道如何進行時, 您可以使用「告訴我搜尋」功能, 快速找出您要尋找的專案。 「操作說明搜尋」了解您正在嘗試完成的作業，並且會提供建議，協助您更快速地完成作業。
  
## <a name="undo-and-redo"></a>復原和取消復原
<a name="bkmk_Undoredo"> </a>

針對使用中的簡報，可無限次復原 (Ctrl + Z) 和取消復原 (Ctrl + Y) 目前編輯工作階段的近期動作。 如果編輯工作階段已逾時，或者您切換到 [讀取檢視] 超過 30 秒，則取消復原的歷程記錄就會重設。 深入瞭解[PowerPoint 中的網頁鍵盤快速鍵](https://go.microsoft.com/fwlink/?LinkId=272950)。
  
## <a name="view-and-add-comments"></a>檢視和新增注釋
<a name="bkmk_Viewaddcomments"> </a>

如果您的檔案儲存在 SharePoint Online 或商務用 OneDrive, 您可以新增、編輯或刪除批註。 
  
如果您的檔案儲存在 OneDrive, 您可以在編輯檢視中新增、編輯或刪除批註, 而且在閱讀檢視中可以查看和更新批註。
  
## <a name="view-and-edit-slide-notes"></a>檢視和編輯投影片備註
<a name="bkmk_Viewslidenotes"> </a>

使用 PowerPoint 網頁版時, 每張投影片的附注都可以顯示或隱藏。 您也可以在 [編輯] 模式中新增注釋。
  
## <a name="wysiwyg-viewing"></a>WYSIWYG 的檢視
<a name="bkmk_WYSIWYGViewing"> </a>

在列印或顯示為完成的產品時，在極類似其外觀的表單中編輯您的簡報。

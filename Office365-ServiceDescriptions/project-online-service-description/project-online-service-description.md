# <a name="project-online-service-description"></a>Project Online 服務說明

Project Online 是適用於專案組合管理 (PPM) 和日常工作彈性線上解決方案。 Project Online 是透過 Office 365 所交付，提供功能強大的專案管理功能，以規劃、設定優先順序以及管理專案和專案組合投資 (幾乎可以從任何位置在任何裝置上使用)。 系統管理員、組合管理員和檢視者、專案和資源管理員、小組負責人和成員都可以使用 Project Online。 
  
Project Online 提供三種計劃  Project Online 基本版、Project Online 專業版和 Project Online 進階版。
  
若要比較各計劃的功能，請參閱[比較 Project Online 計劃](https://products.office.com/en-us/project/compare-microsoft-project-management-software)。
  
您可以在「Office 365 服務說明」中匯出、儲存和列印頁面。了解如何[匯出多個頁面](https://techcommunity.microsoft.com/t5/Project-Support-Blog/Project-Online-New-ways-to-work-with-Team-Assignments/ba-p/362001?advanced=false&collapse_discussion=true&q=New%20ways%20to%20work%20with%20Team%20Assignments&search_type=thread)。
  
## <a name="project-online-boundaries-and-limits"></a>Project Online 界限和限制

Project Online 服務有一些限制。 如需詳細資訊，請參閱 [Project Online 軟體界限和限制](https://support.office.com/en-us/article/Project-Online-software-boundaries-and-limits-5A09DBCE-1E68-4A7B-B099-D5F1B21BA489)。
  
## <a name="data-backup"></a>資料備份

 Project Online 具有相同資料備份和保留原則 Office 365。 如需詳細資訊，請參閱[資料保留、 刪除及毀損 Office 365 中](https://docs.microsoft.com/office365/securitycompliance/office-365-data-retention-deletion-and-destruction-overview)。
  
## <a name="use-of-sharepoint-online"></a>SharePoint Online 的使用

Project Online 服務需要使用 SharePoint Online，該服務將作為 Project Online 服務的一部分提供給您。 提供與 Project Online 專業版或進階版的 SharePoint Online 功能的權限僅限於儲存及存取資料以支援 Project Online。
  
## <a name="licensing-considerations"></a>授權考量

### <a name="internal-users"></a>內部使用者

Project Online 網站上的任何互動需要 Project Online 授權。
  
- Project Online 基本版是已經在使用 Project Online 專業版或 Project Online 進階版中組織的客戶的小組成員的附加元件授權。
    
- 對於更進階的使用情況，使用者必須擁有適當的授權。 專案經理至少需要 Project Online 專業版授權。 產品組合經理需要 Project Online 進階版授權。
    
> [!IMPORTANT]
>  Project Online 基本版是有 Project Online 專業版或 Project Online 進階版的客戶的小組成員*的附加元件*授權。 
<br/>當您最後的 Project Online 專業版或 Project Online 進階版授權到期時，您的 Project Web App 執行個體將被刪除之後 120 天的時間。 
<br/>適用於 Project Online 的試用訂閱，您的 Project Web App 執行個體將會刪除您的試用期結束之後的 30 天。 
  
### <a name="external-users"></a>外部使用者

外部使用者的 project Online 授權將遵循，供內部使用者相同的原則。 Project Online 網站上的任何互動需要 Project Online 授權。

# <a name="project-online-service-description"></a>Project Online 服務說明

Project Online 是 project portfolio management (PPM) 和日常工作的彈性線上解決方案。傳遞到 Office 365、 Project Online 提供強大的專案管理功能的規劃、 排定優先順序、 及管理專案和專案公事包投資 — 從幾乎任何地方幾乎任何裝置上。Project Online 可供系統管理員、 產品組合經理和檢視者、 專案經理和資源管理員、 和小組負責人和成員。 
  
Project Online 提供三種計劃  Project Online 基本版、Project Online 專業版和 Project Online 進階版。
  
若要比較各計劃的功能，請參閱[比較 Project Online 計劃](https://products.office.com/en-us/project/compare-microsoft-project-management-software)。
  
您可以在「Office 365 服務說明」中匯出、儲存和列印頁面。了解如何[匯出多個頁面](https://go.microsoft.com/fwlink/?LinkId=403349)。
  
## <a name="project-online-boundaries-and-limits"></a>Project Online 界限和限制

Project Online 服務有一些限制。如需詳細資訊，請參閱 [Project Online 軟體界限和限制](https://support.office.com/en-us/article/Project-Online-software-boundaries-and-limits-5A09DBCE-1E68-4A7B-B099-D5F1B21BA489)。
  
## <a name="data-backup"></a>資料備份

 Project Online 根據 SharePoint Online 並且遵循與 SharePoint 相同的資料備份原則。如需詳細資訊，請參閱 [SharePoint Online 服務描述](https://technet.microsoft.com/en-us/library/sharepoint-online-service-description.aspx)的資料備份區段。
  
## <a name="use-of-sharepoint-online"></a>SharePoint Online 的使用

您為 Project Online 服務需要 SharePoint Online、 其在佈建為您的 Project Online 服務的一部分的使用。提供與 Project Online Professional 或高階的 SharePoint Online 功能的權限僅限於儲存以及存取支援 Project Online 的資料。
  
## <a name="licensing-considerations"></a>授權考量

### <a name="internal-users"></a>內部使用者

Project Online 網站上的任何互動要求 Project Online 授權。
  
- Project Online 基本版是最低要求。
    
- 更進階的使用情況下，使用者需要有適當的授權。專案經理需要至少 Project Online Professional 授權。產品組合經理需要 Project Online Premium 授權。
    
> [!IMPORTANT]
>  Project Online Essentials 為 Project Online Professional 或 Project Online 高階的客戶的小組成員*的附加元件*授權。<br/>最後一個 Project Online Professional 或 Project Online Premium 授權過期，將會 120 天後刪除 Project Web App 執行個體。<br/>Project Online 試用版訂閱的 Project Web App 執行個體將會刪除 30 天試用版的期間結束之後。 
  
### <a name="external-users"></a>外部使用者

Project Online 授權的外部使用者依下列方式，供內部使用者相同的原則。Project Online 網站上的任何互動要求 Project Online 授權。
